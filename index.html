3-Statement Financial Model & Forecast Analysis Using Excel

This project consists of a fully integrated 3-Statement Financial Model designed to analyze historical performance (2014–2016) and project future financial health (2017–2021). The model links the Income Statement (P&L), Balance Sheet (BS), and supporting schedules (Debt and Fixed Assets) to create a dynamic tool for valuation and strategic decision-making. It includes scenario-based forecasting to assess various business outcomes.

Saldi Financial Case Study: Bookkeeping & Reporting

This project involves building a comprehensive financial reporting model for a company called Saldi. Starting from a historical Balance Sheet (Dec 31, 2014), the objective was to record a series of complex fiscal 2015 transactions using double-entry bookkeeping and ultimately generate a full set of financial statements, including the T-Accounts, Income Statement (P&L), and Closing Balance Sheet for 2015.The project demonstrates the full accounting cycle: from transaction analysis to ledger posting and final reporting.
